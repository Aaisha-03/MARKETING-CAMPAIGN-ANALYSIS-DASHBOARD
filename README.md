MARKETING CAMPAIGN ANALYSIS DASHBOARD :
This dashboard provides a consolidated view of marketing campaign performance across time and channels.

It compares ad spend with key outcomes such as conversions, clicks, and revenue to measure effectiveness.

KPIs like ROAS, CTR, conversion rate, and CPM help evaluate the efficiency of each campaign.

Visuals such as line charts, bar charts, and a donut chart show trends, channel-wise spend, and engagement.

Overall, the dashboard supports data-driven decisions for budget allocation and campaign optimization
